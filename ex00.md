# Install Microchip VSCode in ubuntu 22.04.5

It semms that VSCode DFPs need to open MPLAB_X_IDE project and install them.<br>
Thus, <br>
install MPLAB_X_IDE, <br>
create a blank project, <br>
install VSCode and extension, <br>
open MPLAB_X_IDE project to install DFP. <br>
<br>
感覺,灌VSCode with Microchip extension, 有點運氣運氣的, <br>
vmware SL大法, 經驗是先灌MPLAB_X_IDE, 再灌VSCode, 伴隨多次手動重新啟動,觸發VSCode安裝相關軟件,應該灌得起來.<br>
必要時, 讓VSCode開MPLAB_X_IDE的案子,會載入相關DFP軟件.<br>
EVK is APP-MASTERS25-1<br>
MCU is PIC32CM3204GV00048.<br>
<br>![pic](pic00/evk_s.jpg)<br>
